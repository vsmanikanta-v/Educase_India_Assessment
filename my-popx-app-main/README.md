# PopX Mobile App Clone

This project is a pixel-perfect, mobile-first web application built with **React, Vite, and Tailwind CSS**.  
It replicates the user interface and navigation flow of the PopX application design, featuring screens for **Landing, User Login, User Signup, and Account Settings**.

---

## 🚀 Tech Stack
- **Framework:** React  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Routing:** React Router DOM  

---

## ✨ Features
- **Pixel-Perfect UI**: Designed to closely match the Adobe XD design.  
- **Mobile-First Design**: Layout is centered and framed like a mobile phone screen for a native app feel.  
- **Client-Side Routing**: Smooth navigation between screens without page reloads.  
- **Component-Based Architecture**: Reusable components and modular structure for scalability.  

---

## 🛠 Getting Started

Follow these steps to run the project locally.

### ✅ Prerequisites
Make sure you have the following installed:
- **Node.js v22.x or higher**  
- **npm v10.x or higher**  

Check your versions:
```bash
node -v
npm -v
```

---

### 📥 Installation

1. **Clone the repository**
```bash
git clone https://github.com/your-username/my-popx-app.git
```

2. **Navigate to the project directory**
```bash
cd my-popx-app
```

3. **Install dependencies**
```bash
npm install
```

---

### ▶️ Running the Application

Start the local development server:
```bash
npm run dev
```

The app will be available at:  
👉 [http://localhost:5173](http://localhost:5173) (or the next available port)
